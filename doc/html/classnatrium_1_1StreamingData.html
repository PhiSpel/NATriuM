Detailed Description

template<int dim>
class natrium::StreamingData< dim >

Abstract class to store global streaming data, like e.g. the particle distributions.

Note:
The data to store differs for different approaches of solving the DBE. The only common data which will appear in every method is the particle distribution functions and some global system matrix.
Template Parameters:
dimThe dimension of the flow (2 or 3).
