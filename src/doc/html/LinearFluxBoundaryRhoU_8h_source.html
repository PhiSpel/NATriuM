00016 /*
00017  *                      @short Boundary with fixed velocity (u) and density (\f[ \rho \f]).
00018  *                      This boundary condition bounces back the nonequilibrium parts of the distribution
00019  *                      function, where the equilibrium distribution is constructed with the target values.
00020  *                      For outgoing particle distribution functions the fluxes are set to 0.
00021  *                      For incoming particle distributions fluxes are set to
00022  *                      \f[ f_{\alpha} - f^{+}_{\alpha} = f_{\alpha} - f_{\alpha^{*}} - 2w_{\alpha} \rho_{0} (e_{\alpha}\cdot u_{b})/c^{2}_{s}\f]
00023  *                      This formula shows that we have to couple only opposite distribution functions (COUPLE_ONLY_OPPOSITE_DISTRIBUTIONS)
00024  *                      at single points (COUPLE_ONLY_SINGLE_POINTS - as there are no gradients to be computed)
00025  *                      @note: The boundary has been proposed by Ladd in the standard LBM and has also been used by Min and Lee (2011)
00026  *                      when proposing the SEDG-LBM. It has been shown practically that it  supports the exponential convergence of the scheme.
00027  */
