Detailed Description

template<size_t dim>
class natrium::BoundaryCollection< dim >

The BoundaryCollection class defines all boundaries of a flow domain. Internally, the boundaries are stored in two different std::map, one for the periodic boundaries and one for the non-periodic ones. Its keys are the boundary indicators (for periodic boundaries: the first boundary indicator).

Add a boundary to the flow definition. This definition of addBoundary applies to periodic boundaries. As periodic boundaries have two boundary indicators, they are added twice to the boundary map. Internally, they are additionally added to the vector periodicBoundaries, which can be accessed separate from non-periodic boundaries.
