Detailed Description

template<size_t dim>
class natrium::CFDSolver< dim >

The central class for the CFD simulation based on the DBE.

Note
The CFDSolver itself is quite static but it contains interchangeable modules, e.g. for the Boltzmann model or the time integrator. By these means, a variety of different simulation methods can be covered.
Template Parameters
dimThe dimension of the flow (2 or 3).
