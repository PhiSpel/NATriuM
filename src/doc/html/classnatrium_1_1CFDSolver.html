Detailed Description

template<size_t dim>
class natrium::CFDSolver< dim >

The central class for the CFD simulation based on the DBE.

Note:
The CFDSolver itself is quite static but it contains interchangeable modules, e.g. for the Stencil or the time integrator. By these means, a variety of different simulation methods can be covered.
Template Parameters:
dim The dimension of the flow (2 or 3).
Examples:

step-0.cpp.
