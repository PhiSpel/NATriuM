Detailed Description

This class contains the distribution functions. As the zero-velocity particles (f0) can be ignored for streaming, f0 is split from the other distribution functions (fStream) in the implementation. fStream is a block vector, which has the advantage that it can be multiplied with the SystemMatrix. The DistributionFunctions class is designed to mime the behaviour of a vector<distributed_vector> .
