Whereas this class implements the SEDG spatial discretization, the time integration is done by a subclass of TimeIntegrator, e.g. RungeKutta5LowStorage. In other Finite Element schemes, degrees of freedom can belong to different elements (e.g. at corners of elements). In contrast, DG methods have the degrees of freedom belonging to a single element, which can lead to discontinuities at the element faces.
