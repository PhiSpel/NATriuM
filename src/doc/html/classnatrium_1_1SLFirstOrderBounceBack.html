calculate boundary values. This function overrides the virtual function of SLBoundary.

Parameters:
fe_boundary_values An instance of FEBoundaryValues, that stores the flow variables at t-dt (here only the distribution functions)
q_point the local index of the boundary hit point, i.e. its position in the fe_boundary_values
destination defines the degree of freedom and discrete direction that the calculated value is assigned to (usually defines a point close to the boundary)
eps Not used here. Defines the point in time at which the boundary is hit.
t global time. Only relevant for time-dependent boundary conditions.
