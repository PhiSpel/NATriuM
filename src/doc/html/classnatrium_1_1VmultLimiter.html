Detailed Description

Enforces that the result of a sparse matrix-vector multiplication is between the minimum and maximum of the source values. set fi = aij*fj, if (fi > min_{aij>0} fj) and (fi < max{aij>0} fj); = min_{aij>0} fj, if (fi < min_{aij>0} fj); = max_{aij>0} fj, if (fi > max_{aij>0} fj)
