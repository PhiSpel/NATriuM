Detailed Description

The convergence of the NATriuM solver is analyzed by application to the Taylor-Green vortex in 2D (only periodic walls). This script uses a linear scaling (= constant Mach number). Thus, there is a general compressibily error, which destroys the convergence for the finer meshes. To analyze the results, move the table_order.txt and table_results.txt files to NATriuM/src/analysis/convergence_analysis_basic/ and execute the gnuplot scripts.
